AlfaBotInfo v1.1.1

Руководство использования
    СВК:
        Команды для использования:
            /start - первично

    СРКП:
        Команды для использования:
            /logrender : кидает файл рендера *
            /getbanlist: кидает файл забаненых участников
            /getmemberslist: кидает файл обычных участников
            /getadminslist: кидает файл участников с правами админов
            /getfileslist: кидает список файлов используемых в боте(.pdf)
            /getlog: кидает лог-файл *
            /help: кидает список команд (только админы)
            /ref: обновляет файл рендера (применение изменений из googleSheets)
            /cls: чистит консоль *

            * - debug команды, для обычной работы не потребуются

        Добавление информации в бота:
            googleSheets > alfa_datasetv2_data
                Столбец 'content' должен содержать в себе информацию/гугл лист (доп клава)
                    Гугл лист используется так: <impList> , в скобках название листа.
                    Когда добавлен лист дублируем его в 'renderList' в книге с данными, добавляем по аналогии с остальными
                    затем пишем боту команду '/ref' и ждем пару секунд, готово изменения применены
                Столбец 'title' должен содержать в себе текст кнопки

                Бот отвечает на сообщение именно по тексту кнопки (ищет ответ, отправляет его, добавляя документы и видео при наличии)

                Столбец 'file' содержит в себе полное название файла (если таковой имеется)
                    Если в нем содержится что-либо, вместе с ответом кинет файл(загружается в реальном времени поэтому может потребоваться несколько секунд)

                Столбец 'media' содержит в себе ID видео которое следует отправить вместе с ответом
                    Для получения ID файла нужно отправить видео боту (просто видео ничего писать не надо), бот ответом кинет ID которое мы пишем в лист
                    сделано для оптимизации получения видео т.к. в реальном времени грузится от 30-200 сек.

        Листы:
            MainMenuList: Главная клавиатура, первое что видит пользователь (Button_title: Название кнопки;	Button_Content: то что вернет бот)

            productList	клавиатура: продукты_банка
            nineList	клавиатура: 911
            programsList	клавиатура: рабочее ПО свк
            processingList	клавиатура: процессинг [УП, УП день в день, Ручная выдача]
            deliveryList	клавиатура: Выдачи [Заявки, Кросс, Селфи]
            debetcardList	клавиатура: продукты_банка ->  дебетовые карты/стикеры
            creditcardList	клавиатура: продукты_банка -> кредитные карты/счета
            packetsList	клавиатура: пу_альфы
            chatsList	клавиатура: чаты для свк
            scriptsList	клавиатура: скрипты для СВК
            delBidList	клавиатура: выдачи -> заявки
            delCrossList	клавиатура: выдачи -> кросс продукты
            mailList	клавиатура: почтовые адреса
            numsList	клавиатура: номера телефонов
            impList	клавиатура: важное

            renderList: Лист с данными всех используемых клавиатур в боте (List_name: Название клавиатуры для бота; описание понятно что такое)
            В этот лист добавляем все клавиатуры которые будут использоваться (<*****List>) описание не обязательно, без добавления в этот лист
            клава не подгрузится
    Форматирование текста:
        markdown:
            ` ТЕКСТ ` - текст для копирования
            * ТЕКСТ * - жирный текст
            _ ТЕКСТ _ - курсив
        HTML:
            <s> ТЕКСТ </s> - зачеркнутый
            <u> ТЕКСТ </u> - подчеркнутый
            <i> ТЕКСТ </i> - курсив
            <b> ТЕКСТ </b> - жирный
